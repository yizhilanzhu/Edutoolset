# Edutoolset
### 作为电教委，你在工作中是不是经常遇到以下问题：
- 各种软件东找一个西找一个，毫无章法
- 为了系统优化到处找工具，忙得不可开交
- 历经九九八十一难总算下到工具软件，到用时却不知已经飞到哪个文件夹

Edutoolset正是因为作者经历过这些令人抓狂的事情，所以才决定，将这些教学环境中经常用到的工具软件整理成工具集公开出来
**作为一个工具集，它有以下优点：**

- 分为软件和工具两大模块，每个模块都细分到位
- 软件涉及面广，基本覆盖了教学环境的大部分软件，并且加入一些浏览器插件，以加强浏览器的使用体验
- 工具种类多，涵盖磁盘整理、系统优化等多个方面
- 同时加入一些壁纸，供各位美化一体机桌面

**那么这么厉害的工具集应该在哪里下载呢？**
目前Edutoolset仍在不断完善，因此分发策略如下：
1. 主更新在huang1111网盘，以文件夹的形式存放，不想完整下载整个Edutoolset的，可以在此选择性的下载其中的软件工具
2. 不定期将Edutoolset整个打包成7z压缩包，放在123云盘及希沃品课个人云盘上进行分流，整体打包压缩的Edutoolset将会有版本号（从1.0开始），但新版本发布时会删除旧版本资源，以空出网盘空间，并不会保留旧版本
~~别问为什么，问就是希沃品课只有5GB存储空间~~
3. 由于希沃品课限制1GB+文件上传，因此希沃品课版本将做成7z分卷压缩包
